# Workflow: Release Checklist

## Objective
Walk through all steps to release a track or project from finished mix to live on platforms.

## Inputs Required
- Track title
- Target release date
- Distribution platform (DistroKid, TuneCore, etc.)

## Steps
1. Finalize mix and master — confirm stems are backed up to Drive
2. Export WAV (24-bit, 44.1kHz) and MP3 (320kbps)
3. Create/update artwork (3000x3000px, RGB, JPG)
4. Upload to distributor — set release date, add metadata (ISRC, genre, mood)
5. Submit for playlist consideration (Spotify for Artists, Apple Music for Artists)
6. Draft release announcement (social posts, email if applicable)
7. Confirm live on release date

## Edge Cases
- If distributor rejects artwork: check pixel dimensions and color profile
- If release date needs to move: update in distributor dashboard ASAP (some need 7+ days lead time)
