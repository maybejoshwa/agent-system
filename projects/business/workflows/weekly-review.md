# Workflow: Weekly Review

## Objective
10-minute structured review of open work, pipeline, and priorities to start or close out the week.

## Inputs Required
- None — pulls live data from Gmail and Drive

## Steps
1. Run `/inbox-summary` — surface any action items from email
2. Run `/drive-recent` — check for any docs that need attention
3. Review active ventures list in `CLAUDE.md` — update status if anything changed
4. Identify top 3 priorities for the week
5. Note any blockers or decisions needed

## Edge Cases
- If inbox is overwhelming: focus only on emails from the last 48 hours
- If priorities conflict: flag for explicit decision rather than guessing
