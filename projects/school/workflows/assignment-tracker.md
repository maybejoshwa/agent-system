# Workflow: Assignment Tracker

## Objective
Break down a new assignment into clear steps and get it on the calendar.

## Inputs Required
- Assignment name and description
- Due date
- Course name

## Steps
1. Clarify the deliverable — what exactly is being submitted?
2. Identify any research, reading, or prep work needed first
3. Break into 3-5 concrete subtasks with estimated time per task
4. Add due date to Google Calendar via `/calendar-brief` or `gws calendar events insert`
5. Create a working doc in Google Drive to draft in

## Edge Cases
- If due date is less than 48 hours away: skip planning, go straight to execution
- If assignment requirements are unclear: note the ambiguity and ask before starting
